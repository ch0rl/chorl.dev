“The PACE codes” are codes of practice published in accordance with the Police and Criminal Evidence Act 1984. Most codes contain paragraphs requiring they be made available in police stations; however, in my experience, they are not. These provisions are given below for reference.

Code Ref Quote
A 1.1 This code of practice must be readily available at all police stations for consultation by police officers, police staff, detained persons and members of the public.
B 2.1 This Code must be readily available at all police stations for consultation by:
  • police officers
  • police staff
  • detained persons
  • members of the public
C 1.2 This Code of Practice must be readily available at all police stations for consultation by:
  • police officers;
  • police staff;
  • detained persons;
  • members of the public.
D 2.1 This Code must be readily available at all police stations for consultation by:
  • police officers and police staff
  • detained persons
  • members of the public
G 1.5 This Code of Practice must be readily available at all police stations for consultation by police officers and police staff, detained persons and members of the public.
H 1.8 This Code of Practice must be readily available at all police stations for consultation by:
  • police officers;
  • police staff;
  • detained persons;
  • members of the public.
I 1.7 This Code of Practice must be readily available at all police stations for consultation by:
  • police officers;
  • police staff;
  • detained persons;
  • members of the public.
E 1.1 This Code of Practice must be readily available for consultation by:
  • police officers;
  • police staff;
  • detained persons;
  • members of the public.
F 1.1 This Code of Practice must be readily available for consultation by police officers and other police staff, detained persons and members of the public
