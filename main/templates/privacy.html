Privacy Information

All the following information can be deduced from this site’s source code.

When one makes a request to any resource under www.chorl.dev, the following are recorded; in general, see Nginx: Module ngx_http_core_module, Variables.

  • “$request_uri”: the path of the request.
  • “$remote_addr”: the internal IP address of the request.
  • “$time_iso8601”: the date and time of the request.
  • “$http_user_agent”: the client’s user-agent.
  • “$uid_got” and “$uid_set”: the internal user-id of the client, for tracking sessions.
  • “$request”: the request line (i.e., “GET / HTTP/1.1”).
  • “$status”: the status of the request.
  • “$http_x_forwarded_for”: the original IP address of the request.
  • “$http_referer”: the value of the “referer” header.

This data is then forwarded to an external database and stored indefinitely.
